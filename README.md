# Trader Behaviour vs Market Sentiment Analysis

This project analyzes how cryptocurrency market sentiment influences trader performance and behaviour using Hyperliquid trading data and the Bitcoin Fear & Greed Index.

Developed as part of the Data Science Intern assignment for Primetrade.ai.

## Objective
To evaluate how market sentiment impacts trader profitability, risk-taking behaviour, and trading activity.

## Dataset
- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trading Data

## Methodology
- Cleaned and merged time-series trading data with market sentiment
- Converted UNIX timestamps and aligned datasets at daily level
- Computed trader performance and behavioural metrics
- Compared results across sentiment regimes

## Key Findings
- Highest trader profitability observed during Greed sentiment periods
- Larger trade sizes observed in Fear and Extreme Greed regimes
- Market sentiment significantly influences trader behaviour and performance

## Strategy Implications
- Reduce exposure during extreme sentiment conditions
- Apply sentiment-based position sizing
- Use market sentiment as a risk-adjustment indicator

## Technologies Used
Python, Pandas, Data Analysis, Visualization
